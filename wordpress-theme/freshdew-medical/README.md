# FreshDew Medical Clinic WordPress Theme

A modern, responsive WordPress theme designed specifically for FreshDew Medical Clinic with full hospital management system integration.

## Features

- ✅ Modern, responsive design
- ✅ Custom page templates (Home, Contact, etc.)
- ✅ Integrated contact information
- ✅ OpenStreetMap integration
- ✅ Hero section with image support
- ✅ Customizable navigation menus
- ✅ Footer with contact details
- ✅ Mobile-friendly design

## Installation

1. Upload the theme folder to `/wp-content/themes/`
2. Activate the theme in WordPress admin under `Appearance > Themes`
3. Configure navigation menu under `Appearance > Menus`
4. Create pages and assign templates as needed

## Hero Image

Add the fresh leaf hero image to `/assets/images/fresh-leaf-hero.jpg`. If the image is not available, the theme will use a gradient background.

## Customization

### Colors
Edit `style.css` to customize colors and styling.

### Layout
Page templates can be customized in the `templates/` directory.
