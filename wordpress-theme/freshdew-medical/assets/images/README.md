# Images Directory

## Hero Image - Fresh Leaf

Please add the fresh leaf image (with water droplets) to this directory with the filename:
- `fresh-leaf-hero.jpg`

The image should be:
- High resolution (1920x1080 or larger recommended)
- Optimized for web (compressed but maintaining quality)
- Format: JPG or PNG

This image will be used in:
- Hero section on the home page
- About section background
- Other promotional areas

If the image is not available, the theme will fall back to a gradient background.
